#  Smart Study Planner (Java)

##  Overview
The **Smart Study Planner** is a simple Java console-based application that helps students generate an effective study plan based on subject difficulty and available days before exams.

It prioritizes harder subjects and distributes them across study days to ensure better preparation.

---

##  Features
- Accepts multiple subjects as input
- Allows difficulty levels:
  - 1 → Easy  
  - 2 → Medium  
  - 3 → Hard  
- Automatically sorts subjects based on difficulty
- Generates a day-wise study plan
- Ensures all subjects are covered repeatedly if needed

---

##  Technologies Used
- Java
- Collections Framework (`ArrayList`, `Collections.sort`)
- Scanner (for user input)
